# HabitCity prototype

習慣を達成すると建物が少しずつ完成していく、習慣化アプリの試作版です。

## できること
- 初回説明画面
- 習慣の追加
- 毎日の達成チェック
- 全タスク達成で建物が 1 段階進む
- 完成した建物の一覧表示
- 端末内保存（localStorage）

## ファイル
- `index.html` の 1 ファイルで動きます
- GitHub Pages でもそのまま公開できます

## 公開方法
1. GitHub のリポジトリ設定を開く
2. `Pages` を開く
3. `Deploy from a branch` を選ぶ
4. Branch を `main`、Folder を `/root` ではなく `/habitcity` は選べないため、公開する場合は `index.html` をリポジトリ直下に移すか、別リポジトリに置く

- 週間カレンダー
